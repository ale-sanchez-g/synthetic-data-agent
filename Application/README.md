# Application

This directory contains the application code for the Synthetic Data Generator Agent.

## Structure

- **lambda-functions/**: AWS Lambda functions for data generation, schema validation, and quality metrics
- **api/**: API specifications and endpoint definitions (OpenAPI/FastAPI)
- **utils/**: Utility functions and helper modules
- **schemas/**: JSON schemas and data type definitions

## Purpose

The Application folder houses all the runtime code for the synthetic data generation system, including:
- Data generation logic using Faker library
- Schema validation and compliance checking
- Quality metrics calculation
- API endpoints for client interaction
