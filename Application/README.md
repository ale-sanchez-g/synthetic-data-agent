# Application

This directory contains the application code and logic for the Synthetic Data Generator Agent.

## Purpose

Houses the core application components including:
- Lambda function code for data generation
- API implementations
- Business logic and data generation algorithms
- Schema validation logic
- Compliance checking rules
- Data quality metrics calculators

## Structure

Application code will be organized by component:
- **Lambda Functions**: Serverless functions for AWS Bedrock Agent action groups
- **API Handlers**: FastAPI endpoints for data generation requests
- **Core Logic**: Data generation engines and utilities
- **Models**: Data models and schema definitions
